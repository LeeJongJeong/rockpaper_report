# 락페이퍼 주간업무보고 분석 대시보드

## 프로젝트 개요
락페이퍼 주간업무보고 엑셀 파일(.xlsx)을 브라우저에서 직접 업로드하여 **인터랙티브 데이터 분석 대시보드**를 생성하는 단일 HTML 파일 기반 웹 애플리케이션입니다.

모든 처리는 **100% 클라이언트(브라우저) 내**에서 수행되며, 서버 전송 없이 수만~수십만 행의 데이터를 효율적으로 분석합니다.

---

## 주요 기능

### 📂 파일 업로드
- 드래그앤드롭 또는 파일 선택 버튼
- .xlsx 파일 자동 파싱 (SheetJS)
- 로딩 오버레이 + 에러 토스트 알림
- 다른 파일로 교체 기능

### 📅 날짜 범위 필터
- Flatpickr 기반 날짜 선택
- 데이터 내 min/max 자동 감지
- 전체 시각화 실시간 연동

### 🔍 다중 필터 (AND 로직)
| 필터 | 대상 컬럼 |
|------|-----------|
| 부서 | 부서명 |
| 엔지니어 | 엔지니어 |
| 제품 | 제품명 |
| 지원유형 | 지원유형 |

| 담당영업 | 담당영업 |

- 체크박스 드롭다운 + 전체선택/해제
- 필터 요약 배지 표시
- 150ms 디바운스 적용

## 6개 분석 탭

### 1️⃣ Overview
- **7개 KPI 카드**: 총 지원 건수, 활동 엔지니어, 지원 고객사, 지원 제품, 데이터 기간, 일 평균 건수, **전체 평균 가동률**
- **일별 지원 건수 추이** (라인차트)
- **제품별 지원 비중** (도넛차트)
- **지원유형별 건수** (가로 막대차트)
- **부서별 지원 건수** (막대차트)
- **요일별 지원 패턴** (막대차트 - 주말 강조)
- 팀별(부서별) 가동률 현황 막대차트 (DS 1T / DS 2T / 전체 평균)

### 2️⃣ 엔지니어 분석 ⭐ (v3 가동률 추가)
- **5개 KPI**: 총 엔지니어, 총 투입시간, **평균 가동률(%)**, 외부지원 비율, 최다 활동 엔지니어
- **업무 부하 버블차트**: X=건수, Y=시간, 크기=고객사수 → 부하 분포 시각화
- **엔지니어별 가동률 막대차트**: 활동일×8h 기준, 색상 등급(🏢 80%↑ / 🏡 60~80% / 🏠 60%↓)
- **팀별(부서별) 가동률 비교 차트**: DS 1T vs DS 2T + 가동시간 라인
- **역량 레이더차트 (Top 5)**: 건수·시간·고객사·제품·활동일 5축 비교
- **내부/외부 비율 스택바**: 엔지니어별 업무 구성 파악
- **투입시간 비교 (부서 색분류)**: DS 1T(파랑) vs DS 2T(보라) 시각적 구분
- **일별 지원 추이 (Top 6)**: 주요 엔지니어 활동 패턴 분석
- **제품 전문성 스택바**: 엔지니어별 지원 제품 다양성
- **엔지니어 × 지원유형 히트맵**: 업무 유형 전문화 파악
- **종합 스코어카드**: 건당시간, **가동시간/소정근무/가동률(%)**, 업무강도(h/일) 지표 포함

### 3️⃣ 제품 분석 ⭐ (v2 재설계)
- **5개 KPI**: 지원 제품 수, 총 투입시간, 제품당 평균, 연관 고객사, 단일엔지니어 제품(리스크)
- **포트폴리오 맵 버블**: X=고객사수, Y=건수, 크기=엔지니어수
- **제품군별 비중 도넛**: MySQL/MariaDB/PostgreSQL/NoSQL/클라우드 DB
- **건수 vs 투입시간 이중축**: 건당 소요시간 효율 분석 (막대+라인)
- **지원유형 구성 스택바**: 제품별 기술지원/점검/Presales/내부 비중
- **부서별 투입 비중**: DS 1T vs DS 2T 제품별 리소스 배분
- **일별 추이 (Top 6)**: 제품별 지원 트렌드
- **제품 × 엔지니어 히트맵**: 커버리지 공백 식별
- **상세 스코어카드**: 건당시간, DS1T:2T 비율, 엔지니어 의존도

### 4️⃣ 지원유형 분석 ⭐ (v2 재설계)
- **4개 KPI**: 지원유형 종류, 외부 고객지원, 방문 지원, 건당 평균시간
- **업무 대분류 도넛**: 7대 분류 비중
- **방문 vs 원격 파이**: 이동 비용 분석
- **건수 vs 투입시간 이중축**: 유형별 효율성 비교
- **대분류별 일별 추이**: 업무 패턴 변화 감지
- **부서별 업무유형 비중**: 부서 간 역할 차이 분석
- **엔지니어 투입수 분석**: 유형별 인력 분산/집중도
- **지원유형 × 제품 히트맵**: 제품-업무 조합 분석
- **상세 스코어카드**: 방문/원격 구분, 건당시간, 고객사수 포함

### 5️⃣ 고객사 분석 ⭐ (v2 재설계)
- **5개 KPI**: 지원 고객사, 총 투입시간, 단일엔지니어 고객(리스크), 멀티제품 고객, 최다 지원 고객
- **세그멘테이션 버블**: X=건수, Y=제품수, 크기=엔지니어수
- **Top 15 지원유형 구성**: 고객별 기술지원/점검/Presales 비중
- **건수 vs 투입시간 이중축 (Top 15)**: 고객별 서비스 효율
- **Top 5 지원 추이**: 주요 고객 일별 활동
- **담당영업별 현황**: 건수 + 고객사수 + 투입시간
- **인력 리스크 분석**: 🔴1명=위험/🟡2명=주의/🟢3명+=양호
- **고객사 × 제품 히트맵**: 제품 포트폴리오 매핑
- **종합 스코어카드**: 건당시간, 인력 리스크 뱃지

### 6️⃣ 지원내역 상세
- **필터 연동 테이블**: 전체 데이터 확인
- **검색**: 실시간 텍스트 검색
- **정렬**: 컬럼 헤더 클릭 정렬 (ASC/DESC)
- **페이지네이션**: 50/100/200행 선택
- **엑셀 다운로드**: 필터 적용된 데이터 내보내기

## 사용 방법
1. `index.html`을 브라우저에서 열기
2. 락페이퍼에서 다운받은 주간업무보고 엑셀 파일(.xlsx)을 드래그앤드롭 또는 파일 선택
3. 자동 파싱 후 대시보드가 렌더링됨
4. 날짜/필터를 조정하여 원하는 관점의 분석 수행
5. 탭 전환으로 다양한 분석 뷰 확인
6. 필요 시 지원내역 상세 탭에서 엑셀 다운로드

### 핵심 변경: `calcHours()` 함수에 점심시간 1시간 차감

- `TABLE_COLUMNS` 배열 기반이므로 정렬/검색/엑셀 내보내기 모두 자동 지원

## v5 변경 사항 (2026-02-28) - 제품군 분류 동적 처리
### 핵심 변경: `productGroupOf()` 하드코딩 → 확장형 동적 분류

#### 기존 문제
