# 락페이퍼 주간업무보고 분석 대시보드

브라우저에서 `주간업무보고(.xlsx/.xls)` 파일을 바로 업로드해 분석하는 **단일 HTML 대시보드**입니다. 서버 없이 클라이언트에서 파싱/집계/시각화를 수행합니다.

---

## 핵심 특징

- **무설치/무빌드**: `index.html` 하나로 실행
- **클라이언트 전용 처리**: 엑셀 데이터 서버 전송 없음
- **대용량 대응**: 디바운스, 단일 pass 집계, 차트 인스턴스 재사용, 레이지 탭 렌더링
- **다중 파일 병합 지원**: 파일 추가 업로드 후 통합 분석

---

## 기술 스택

| 라이브러리 | 용도 |
|---|---|
| SheetJS (xlsx) | 엑셀 파싱 |
| Chart.js | 차트 시각화 |
| Flatpickr | 날짜 범위 선택 |
| Font Awesome | 아이콘 |
| Google Fonts (Inter, Noto Sans KR) | 폰트 |

---

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. 주간업무보고 파일(`.xlsx`/`.xls`)을 드래그앤드롭 또는 파일 선택으로 업로드합니다.
3. 로딩 후 날짜/필터/탭으로 분석합니다.
4. 상세 탭에서 검색·정렬·페이지네이션·엑셀 내보내기를 사용합니다.

> 별도 서버/빌드 명령이 필요하지 않습니다.

---

## 입력 데이터 형식

### 필수 컬럼

- `작업시작일시`
- `작업종료일시`
- `부서명`
- `엔지니어`
- `제품명`
- `지원유형`
- `고객사명`
- `지원내역`
- `지원도시`
- `담당영업`

### 파일/파싱 규칙

- 지원 확장자: `.xlsx`, `.xls`
- 파일 크기 제한: **200MB**
- 기본 파싱 기준: **헤더 3행, 데이터 4행부터**

---

## 필터 기능

- 날짜 범위 필터 (min/max 자동 감지)
- 6개 다중 필터(AND): 부서 / 엔지니어 / 제품 / 지원유형 / 고객사 / 담당영업
- 드롭다운 체크박스, 검색, 전체선택/해제
- 필터 적용 디바운스: **150ms**
- 차트 클릭 기반 드릴다운 필터 지원

---

## 분석 탭 (총 7개)

1. **Overview**
   - 핵심 KPI, 일별 추이, 제품/지원유형 비중, 부서/요일 패턴, 팀 가동률
2. **엔지니어 분석**
   - 가동률·업무부하 버블·내/외부 구성·전문성·히트맵·스코어카드
3. **제품 분석**
   - 제품 포트폴리오, 효율(건수 vs 시간), 부서 투입 비중, 커버리지
4. **지원유형 분석**
   - 업무 대분류, 방문/원격, 유형별 효율, 유형×제품 히트맵
5. **고객사 분석**
   - 고객 세그멘테이션, 인력 리스크, 고객×제품 커버리지
6. **담당영업 분석**
   - 영업별 포트폴리오, 건수/시간/유형 구성, 고객/엔지니어 분포
7. **지원내역 상세**
   - 필터 연동 테이블, 검색/정렬/페이지네이션, 엑셀 다운로드

---

## 주요 계산/집계 로직

- `calcHours(row)`
  - 기본: `작업종료일시 - 작업시작일시`
  - 예외: `09:00~18:00` 또는 `08:30~17:30`은 점심 1시간 차감
- `isBillable(type)`
  - 가동률 포함 항목(기술지원/점검지원/Presales/비상대기/현장실습/고객사교육)
- 가동률 공식
  - `billableHours / (activityDays × 8) × 100`
- 집계 방식
  - 엔지니어/제품/고객사/담당영업 단위 **단일 pass 집계**
